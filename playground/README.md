EiderScript Playground is a dedicated development environment for testing EiderScript YAML manifests with live compilation and real-time reactivity.
</p>

## **Prerequisites**

- **Node.js**: >= 22
- **pnpm**: >= 9

## **Installation**

From the root directory, navigate to the playground and install the dependencies:

```bash
cd playground
pnpm install
```

## **Development**

To start the playground in development mode:

```bash
pnpm dev
```

The application will be available at `http://localhost:5173`.

## **Features**

- **Live Compilation**: Instant feedback as you write EiderScript YAML.
- **Syntax Highlighting**: Real-time YAML highlighting powered by Prism.js.
- **Example Library**: Instant access to standardized EiderScript examples.
- **Reactive Preview**: View rendered Vue components with full reactivity.
